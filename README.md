# studyhub

[![Live Demo](https://img.shields.io/badge/🚀_Live_Demo-Visit-blue?style=for-the-badge)](https://mayank-dev-15.github.io/studyhub)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Language](https://img.shields.io/badge/Language-JavaScript/Python-green)

Student portal with course management, assignments, grades tracking, and admin panel.

`education` `student-portal` `courses` `grades`

---

## ✨ Features

- Course catalog and enrollment
- Assignment submission and tracking
- Grade viewing and GPA calculation
- Admin panel for course management
- Student dashboard with progress
- Notifications system
- Search and filter courses
- Responsive design for all devices

## 📦 Installation

```bash
git clone https://github.com/mayank-dev-15/studyhub.git
cd studyhub
```

```bash
cd studyhub
open index.html
# Or serve with Python
python -m http.server 8080
```

---

## 💡 Usage

- Login to access your dashboard
- Browse courses in the catalog
- Submit assignments through the portal
- View grades and GPA
- Admins can manage courses via the admin panel
