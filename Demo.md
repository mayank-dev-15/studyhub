## About

**studyhub** is a student portal with courses & grades.

### Key Features

- Course catalog and enrollment
- Assignment submission and tracking
- Grade viewing and GPA calculation
- Admin panel for course management
- Student dashboard
- Notifications system
- Responsive design

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Python backend

---

## Controls & Usage

| Control | Action |
|---------|--------|
| `Login` | Access dashboard |
| `Courses tab` | Browse courses |
| `Assignments` | View/submit work |
| `Grades` | View grades |
| `Admin` | Manage courses |
